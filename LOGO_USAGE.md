# VEXEL Logo Usage Guide

## Logo Location

The official VEXEL logo is stored in the repository:
- **File:** `assets/vexel-logo.png`
- **Original filename:** `VEXEL Logo_ Modern Minimalist Design_20260122_201608_0000.png`
- **Format:** PNG (500x500px, RGBA)
- **Size:** ~199KB

## GitHub-Hosted URL

For use in markdown files and documentation, use the repository-hosted URL:
```
https://github.com/Violet-Site-Systems/VEXEL/blob/main/assets/vexel-logo.png
```

## Where the Logo Appears

The VEXEL logo has been added to the following documentation files:

1. **README.md** - Main repository README (top of page)
2. **PROJECT_DOCUMENTATION.md** - Project documentation overview (top of page)
3. **CONTRIBUTING.md** - Contributor guidelines (top of page)

## Logo Usage

### In Markdown Files

```markdown
<div align="center">
  <img src="https://github.com/Violet-Site-Systems/VEXEL/blob/main/assets/vexel-logo.png" alt="VEXEL Logo - Digital Identity for LIFE and Beyond" width="400"/>
</div>
```

### Recommended Sizes

- **Documentation headers:** 400px width
- **Social media:** 500x500px (original size)
- **Small icons:** 100-150px width

## Logo Tagline

The logo includes the tagline: **"Digital Identity for LIFE and Beyond"**

This reflects VEXEL's mission to create sovereign, interoperable identity systems that preserve cryptographic sovereignty across distributed ecosystems.

## Brand Guidelines

- The logo features a hexagonal design with gradient colors (purple, orange, pink)
- Primary colors: Purple (#6B46C1), Orange (#FF8C42), Dark background
- Use on dark or light backgrounds with appropriate contrast
- Maintain aspect ratio when resizing
- Do not modify or alter the logo design
