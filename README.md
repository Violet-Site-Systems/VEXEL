## üöÄ Quick Start

### For Phase 1.2 (Database & IPFS)

```bash
# Clone the repository
git clone https://github.com/Violet-Site-Systems/VEXEL.git
cd VEXEL

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env with your database and IPFS credentials

# Run database migrations
npm run build
npm run migrate

# Run example
npm run example

# Run tests
npm test
```

## üèóÔ∏è Architecture

VEXEL implements a multi-layer architecture:

- **Blockchain Layer:** Ethereum/Polygon, Smart Contracts, DID Registry
- **Data Layer:** PostgreSQL, IPFS, Arweave, Subgraph Protocol
- **Application Layer:** DID Integration, HAAP Protocol, Inheritance Engine
- **Bridge Layer:** API Gateway, WebSocket, Cross-Platform Integration
- **User Interface:** Monitoring Dashboard, Real-time Status Feeds
